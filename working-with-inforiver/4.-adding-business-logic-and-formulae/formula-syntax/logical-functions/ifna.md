# IFNA

Return value2, if value1 is not number else returns value1

## Syntax

```javascript
IFNA(value1,value2)
```

## Arguments

value1- number or reference

value2- number or reference

## **Return Value**

Either value1 or value2

## **Example**

```javascript
IFNA((AC-PY)/PY, 0)
```

Returns AC-PY/PY if the value is a number else returns 0
