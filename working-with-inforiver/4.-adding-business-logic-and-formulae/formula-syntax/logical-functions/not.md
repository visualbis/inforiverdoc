# NOT

A NOT statement returns TRUE if the argument is FALSE and FALSE if the argument is TRUE.

## Syntax

```javascript
NOT( logical_test )
```

## **Return Value**

## **Example**

```javascript

```

Will return 50 if the region is not West otherwise returns 25
