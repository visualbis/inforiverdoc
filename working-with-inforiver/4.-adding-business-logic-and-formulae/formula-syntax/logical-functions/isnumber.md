# ISNUMBER

Checks whether value is a valid number

## Syntax

```javascript
ISNUMBER(value)
```

## Arguments

value - number or reference

## **Example**

```javascript
IF(ISNUMBER((AC-PY)/PY), (AC-PY)/PY , 0)
```

Returns AC-PY/PY if the value is a number else returns 0
