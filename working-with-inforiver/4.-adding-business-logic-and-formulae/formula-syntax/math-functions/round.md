# ROUND

The Round function returns the round of a number.

## Syntax

```javascript
ROUND(value, significance)
```

## Arguments

value - number or reference

significance - number

## Example

```javascript
ROUND(COLUMN1, 2)
```

Returns the round of COLUMN1 to the significance of two decimal.
