# CEILING

The Ceiling function returns the ceil of a number.

## Syntax

```javascript
CEILING(value, significance)
```

## Arguments

value - number or reference

significance - number

## Example

```javascript
CEILING(COLUMN1)
```

Returns the ceiling of COLUMN1 to the significance of two decimal.
