# FLOOR

The Floor function returns the floor of a number.

## Syntax

```javascript
FLOOR(value, significance)
```

## Arguments

value - number or reference

significance - number

## Example

```javascript
FLOOR(COLUMN1)
```

Returns the floor of COLUMN1 to the significance of two decimal
