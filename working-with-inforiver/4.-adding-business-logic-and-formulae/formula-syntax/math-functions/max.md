# MAX

The MAX function returns the largest number in a set of values. Node references also accepted as arguments.

## Syntax

```javascript
MAX(value1,[value2]....)
```

## Example

```javascript
MAX(COLUMN1,COLUMN2,COLUMN3)
```

Returns the maximum of COLUMN1, COLUMN2, COLUMN3
