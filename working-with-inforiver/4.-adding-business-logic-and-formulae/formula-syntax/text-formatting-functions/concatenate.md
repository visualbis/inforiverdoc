# CONCATENATE

Concatenates all the arguments

## Syntax

```javascript
CONCATENATE(string1, string2, ..)
```

## Arguments

string1, string2.. - string&#x20;

## Example

```javascript
CONCATENATE("Today's price is ",TEXT(6000,"$###,###.00"))
```

Above formula will return "Today's price is $6,000
