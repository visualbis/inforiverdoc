# Snowflake

Allows you to configure Snowflake as a destination for Inforiver writeback.&#x20;

After navigating to the add destination page, the following details will have to be entered.&#x20;

* Account
* Database
* Schema
* Username
* Password
* Warehouse
* Role
* Encrypted Connection (Y/N)
* Table Name
