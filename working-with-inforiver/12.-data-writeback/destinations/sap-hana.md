# SAP HANA

Allows you to configure SAP HANA as a destination for Inforiver writeback.&#x20;

After navigating to the add destination page, the following details will have to be entered.&#x20;

* Server
* Port
* Schema
* Username
* Password
* Encrypted Connection (Y/N)
* Table Name
