# Amazon Redshift

Allows you to configure Amazon Redshift as a destination for Inforiver writeback.&#x20;

After navigating to the add destination page, following details will have to be entered.&#x20;

* Server
* Port
* Database
* Schema
* Username
* Password
* Encrypted Connection (Y/N)
* Table Name
