# Databricks

Allows you to configure Databricks as a destination for Inforiver writeback.&#x20;

After navigating to the add destination page, the following details will have to be entered.&#x20;

* Host Name
* Warehouse ID
* Access Token
* Catalog
* Schema
* Table Name
