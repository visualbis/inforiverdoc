# Oracle

Allows you to configure Oracle as a destination for Inforiver writeback.&#x20;

After navigating to the add destination page, the following details will have to be entered.&#x20;

* Server
* Service name
* SID
* Port
* Username
* Password
* Encrypted Connection (Y/N)
* Table Name
