# Charts

In the header and footer, you can add all the inline charts such as sparklines, bullets, range plots, dot plots, and other unique charts such as donut charts and progress bars. Let's take a detailed look.&#x20;

## 1. Adding the chart

a) Click on 'Header & footer' in the 'Design' tab.

<figure><img src="../../../.gitbook/assets/8.5.1 Charts.png" alt=""><figcaption><p>Adding a chart</p></figcaption></figure>

b) Choose 'Chart' from the 'Type' option.

<figure><img src="../../../.gitbook/assets/8.5.2 Charts.png" alt=""><figcaption><p>Choosing the 'Chart' element</p></figcaption></figure>

c) By default, a donut chart gets inserted based on the data fields added to the report. Here we have a donut chart for the five subcategories.

<figure><img src="../../../.gitbook/assets/8.5.3 Charts.png" alt=""><figcaption><p>Default chart gets inserted</p></figcaption></figure>

d) You can see the customization options present in the 'Customize' tab. Appearance and Legend config are covered in [customization options](charts.md#4.-customization-options).

<figure><img src="../../../.gitbook/assets/8.5.4 Charts.png" alt=""><figcaption><p>Customization options</p></figcaption></figure>

## 2. Data selection

Note that by default, the table is greyed out as shown in the image.

<figure><img src="../../../.gitbook/assets/8.5.28 Charts.png" alt=""><figcaption><p>Table greyed out</p></figcaption></figure>

a) Click on 'Data selection' in the toolbar. The table is no longer greyed out. You can now select a cell from the table to select data.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.29 Charts.png" alt=""><figcaption><p>Data selection</p></figcaption></figure>

b) Use Ctrl/Shift + select to select multiple cells. The donut chart gets updated as shown.

<figure><img src="../../../.gitbook/assets/8.5.20 Charts.png" alt=""><figcaption><p>Data selection</p></figcaption></figure>

c) Once you have selected the data, click on the 'check mark' icon in the cell editor.

<figure><img src="../../../.gitbook/assets/8.5.21 Charts.png" alt=""><figcaption><p>Data selection</p></figcaption></figure>

d) The changes are saved and the table is again greyed out.

<figure><img src="../../../.gitbook/assets/8.5.22 Charts.png" alt=""><figcaption><p>Data selection frozen</p></figcaption></figure>

## 3. Chart types

All the chart types that are shown in the 'Home' tab can be added to the header. Let's see how to add them.&#x20;

a) In the 'Customize' tab, click on 'Data selection' and select three grand totals. You can see a donut chart as shown in the image.

<figure><img src="../../../.gitbook/assets/8.5.30 Charts.png" alt=""><figcaption><p>Donut chart</p></figcaption></figure>

b) Click on the 'Right arrow' in the cell as shown in the image. Notice that in the toolbar it says 'Donut'.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.31 Charts.png" alt=""><figcaption><p>Changing the chart type</p></figcaption></figure>

c) The bullet chart gets inserted in the cell and you can see the bullet chart variants in the toolbar. Also, let's freeze the data selection by clicking on the 'checkmark' icon as highlighted.

<figure><img src="../../../.gitbook/assets/8.5.32 Charts.png" alt=""><figcaption><p>Bullet chart</p></figcaption></figure>

d) You can use the left and right arrows to switch between the variants or use the dropdown in the toolbar.

<figure><img src="../../../.gitbook/assets/8.5.33 Charts.png" alt=""><figcaption><p>Using the right and left arrows and toolbar</p></figcaption></figure>

e) Another way to select the chart is to use the 'Settings' icon. Select 'Chart types' and you can see a list of the chart types applicable for the data selection. Notice that the 2-value bullets are not shown since we have selected three values.

{% hint style="warning" %}
Some of the variants are not shown here but they can be accessed from the toolbar.
{% endhint %}

<figure><img src="../../../.gitbook/assets/8.5.34 Charts.png" alt=""><figcaption><p>Chart types</p></figcaption></figure>

f) Let's select the stacked bar. Notice that in the toolbar you can see the other chart types such as the dumbbell plot.

<figure><img src="../../../.gitbook/assets/8.5.35 Charts.png" alt=""><figcaption><p>Other variants shown in the toolbar</p></figcaption></figure>

g) From the dropdown, select the 'Dot plot'. The header gets updated.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.36 Charts.png" alt=""><figcaption><p>Choosing the dot plot</p></figcaption></figure>

h) In case two data points are selected, you can also use variants of the progress bar such as conditional bar, angular bar and donut bar.

<figure><img src="../../../.gitbook/assets/8.5.37 Charts.png" alt=""><figcaption><p>Progress bars</p></figcaption></figure>

i) When multiple data points are selected, or rows/columns, the sparkline charts are enabled as shown.

<figure><img src="../../../.gitbook/assets/8.5.38 Charts.png" alt=""><figcaption><p>Sparklines</p></figcaption></figure>

## 4. Customization options

Most of the customization options are already covered in [working with charts](../../6.-working-with-charts.md). The customization options unique to header charts are covered here.&#x20;

### a) All chart types

#### Legend config

By default, the legend is shown on the right of the chart. But you can change the position to the left or turn it off.

<figure><img src="../../../.gitbook/assets/8.5.24 Charts.png" alt=""><figcaption><p>Legend config</p></figcaption></figure>

When the legend is turned off, depending on the available space, the values and percentages are shown as seen in the below image.

<figure><img src="../../../.gitbook/assets/8.5.25 Charts.png" alt=""><figcaption><p>Legend disabled</p></figcaption></figure>



### b) Donut chart

The appearance of the donut chart can be customized by clicking on the 'Appearance' option. You can see the dialog box shown in the image where you can select the color for each arc, arc size and apply a gradient.

<figure><img src="../../../.gitbook/assets/8.5.9 Charts (1).png" alt=""><figcaption><p>Appearance options for donut charts</p></figcaption></figure>

a) When a gradient is applied, you can select the gradient color and the chart gets updated as shown.

<figure><img src="../../../.gitbook/assets/8.5.13 Charts.png" alt=""><figcaption><p>Gradient applied</p></figcaption></figure>

b) When there are more than 5 categories selected, the 'Enable others' option is activated. Note that even though six sub-regions were selected, only five are displayed in the chart.

<figure><img src="../../../.gitbook/assets/8.5.14 Charts.png" alt=""><figcaption><p>Enable others option</p></figcaption></figure>

c) On checking the 'Enable others' checkbox, you can see a segment in the donut called 'Others' which visualizes the sum of the values apart from the top 5 categories. That is, if there are 10 categories added, the last 5 categories will be consolidated and shown as 'Others'.

<figure><img src="../../../.gitbook/assets/8.5.15 Charts (2).png" alt=""><figcaption><p>Others displayed in the chart</p></figcaption></figure>

d) The name 'Others' can be modified by clicking on the 'Pencil' icon and entering in the text field.

<figure><img src="../../../.gitbook/assets/8.5.16 Charts.png" alt=""><figcaption><p>Renaming 'Others'</p></figcaption></figure>

e) The arc size can be customized by using the 'Arc size' field. The available options are Small, Medium, Large and Extra large.

<figure><img src="../../../.gitbook/assets/8.5.17 Charts (1).png" alt=""><figcaption><p>Customizing the arc size</p></figcaption></figure>

### c) Progress bars

Progress bar appearance can be customized as shown below. You can define positive and negative colors using a color picker.

<figure><img src="../../../.gitbook/assets/8.5.39 Charts.png" alt=""><figcaption><p>Appearance</p></figcaption></figure>

### d) Sparklines

In the sparkline customization toolbar, explanations for [Appearance](../../6.-working-with-charts/sparkline-charts.md#viii-appearance), [Null display](../../6.-working-with-charts/sparkline-charts.md#vi-null-display), [Marker](../../6.-working-with-charts/sparkline-charts.md#ii-marker) and [Show label](../../6.-working-with-charts/sparkline-charts.md#viii-show-label) are covered in the '[Working with charts](../../6.-working-with-charts.md)' section.

Let's look at the 'Data config' option.

i) Select the row grand total and choose the sparkline chart type. The area spline chart has been selected here. You can see the sparkline chart in the header as shown below.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.40 Charts.png" alt=""><figcaption><p>Sparkline on selecting a row</p></figcaption></figure>

ii) Click on 'Data config'. In the dialog box, you can see that all the three measures are selected. Let's display only the 2022 Actuals measure in the header. Uncheck the other checkboxes.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.41 Charts.png" alt=""><figcaption><p>Data config options</p></figcaption></figure>

iii) Select a different color using the color picker if needed. The chart gets updated.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.42 Charts.png" alt=""><figcaption><p>Data config options</p></figcaption></figure>

iv) Notice that there is another dropdown 'Measure level selection'. This option is useful when you select a measure. In the below image, we have inserted a sparkline for the 2022 Actuals. By default, the leaf level is considered for the 'Measure level selection' which is sub-category in this case.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.43 Charts.png" alt=""><figcaption><p>Measure level selection</p></figcaption></figure>

v) Each bar in the chart represents a sub-category at a sub-region level. We have turned the markers on for all the points to illustrate this clearly. (Note that due to lack of space, not all the data points are shown).

<figure><img src="../../../.gitbook/assets/8.5.44 Charts.png" alt=""><figcaption><p>Sparkline chart</p></figcaption></figure>

vi) Select 'Sub-region' from the 'Measure level selection' dropdown.

<figure><img src="../../../.gitbook/assets/8.5.45 Charts.png" alt=""><figcaption><p>Measure level selection</p></figcaption></figure>

vii) You can now see the columns for the six sub-regions.

<figure><img src="../../../.gitbook/assets/8.5.46 Charts.png" alt=""><figcaption><p>Sub region sparklines</p></figcaption></figure>

viii) When the 'Measure level selection' is the leaf level, the 'Apply auto sum' option can be used to aggregate the individual values.&#x20;

<figure><img src="../../../.gitbook/assets/8.5.47 Charts.png" alt=""><figcaption><p>Auto sum option</p></figcaption></figure>

ix) The five subcategories are summed across the subregions and shown as columns.

<figure><img src="../../../.gitbook/assets/8.5.48 Charts.png" alt=""><figcaption><p>Auto sum option</p></figcaption></figure>
