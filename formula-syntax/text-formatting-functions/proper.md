# PROPER

The PROPER function converts the first character of a text string into uppercase. The remaining characters will be lowercase. Node references are also accepted as arguments.

## Syntax

```javascript
proper(value:string)
```

## Arguments

value - Required. A text string.

## Example

```javascript
proper('INFORiver')
```
