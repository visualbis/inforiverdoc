# SHOWZEROAS

Use the SHOWZEROAS function to customize how to display zeroes in your reports.

### Syntax

```javascript
SHOWZEROAS.<TYPE>
```

### Arguments

TYPE- Can be BLANK, HYPHEN, CUSTOM, or ZERO. Pass the custom value to use when the TYPE is 'CUSTOM'.

### Example

```
SHOWZEROAS.BLANK
SHOWZEROAS.ZERO
SHOWZEROAS.HYPHEN
SHOWZEROAS.CUSTOM('custom text')
```
