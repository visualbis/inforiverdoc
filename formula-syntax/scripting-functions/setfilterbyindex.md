# SETFILTERBYINDEX

Instead of specifying the filter name, you can set a filter using its index. The first filter that is created will have an index value of 0.

### Syntax

```javascript
SETFILTERBYINDEX(INDEX_VALUE)
```

### Example

```
SETFILTERBYINDEX(3)
```
