# EVEN

The EVEN function rounds off an input number and returns the nearest higher even number. Node references are also accepted as arguments.

## Syntax

```javascript
even(value)
```

## Arguments

value - The input number to be rounded off.

## Example

```javascript
EVEN(40193)
```

Returns 40194.
