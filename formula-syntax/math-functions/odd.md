# ODD

The ODD function rounds off an input number and returns the nearest higher odd number. Node references are also accepted as arguments.

## Syntax

```javascript
odd(value)
```

## Arguments

value - The input number to be rounded off.

## Example

```javascript
ODD(39458)
```

Returns 39459.
