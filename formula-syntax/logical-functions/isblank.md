# ISBLANK

Checks whether value is empty

## Syntax

```javascript
ISBLANK(value)
```

## Arguments

value - number or reference or expression&#x20;

## **Example**

```javascript
ISBLANK(AC)
```

Returns TRUE if AC is empty
