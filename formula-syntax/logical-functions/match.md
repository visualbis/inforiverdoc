# MATCH

Check whether value matches the regex

## Syntax

```javascript
MATCH(value:string,regex:string)
```

## Arguments

value - string

regex - string

## **Example**

```javascript
MATCH(“ABC-123-WEW”,”^ABC”) 
```

Returns TRUE if the string value matches the regex value
